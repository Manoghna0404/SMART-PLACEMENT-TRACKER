# 📖 Overview

The Frontend of Smart Placement Tracker provides an intuitive and responsive user interface for Students and Administrators.

The application enables users to manage placements, applications, online assessments, interview rounds, resume analysis, notifications, and analytics through a modern dashboard experience.
